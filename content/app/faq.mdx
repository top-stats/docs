      </Step>
      <Step>

        ![get-bot-image](/faq/app/get-bot.png)
      </Step>
      <Step>
        Click the command you want to disable, select the **@everyone** role, set it to **‚ùå**, then click **Save**.
        ![edit-permissions-image](/faq/app/edit-permissions.png)
      </Step>
    </Steps>
    <Callout type={'info'}>
        This hides the command for members, but it will remain visible to administrators.
